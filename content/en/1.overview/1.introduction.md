# Keero Bot

Keero Bot is a modular embedded platform designed for AI-driven physical devices. The current hardware revision combines processing, sensing, power management, display support, audio IO, haptics, and module expansion on a single compact board.

The hardware repository shows a clear split between the mainboard and external modules:

- `mainboard/` contains the production board design, fabrication files, 3D models, and schematics.
- `modules/` contains the dock and tracks mechanical assets.
- `assets/` contains hero and PCB render images that are now mirrored into this documentation site for deployment.

## Current Revision

- Project title in schematic: `keero_bot`
- Hardware revision shown in schematic title block: `1.0`
- Schematic date: `2026-03-09`
- PCB export date embedded in repository files: `2026-04-02`

## Platform Goals

- Open and reproducible hardware files
- Modular architecture for future accessory boards
- Battery-powered operation with managed rails
- Support for vision, audio, touch/button interaction, haptics, and motion sensing
- A future firmware stack for AI-driven behavior and external service integration

## Mainboard at a Glance

The current board integrates:

- `ESP32-S3-WROOM-1U-N16R8` as the central MCU module
- `AXP2101` PMIC for battery charging and rail generation
- `LIS2DW12TR` accelerometer
- `ICS-41350` digital microphone
- `MAX98357AETE+T` I2S speaker amplifier
- `DRV2605LDGSR` haptic driver
- `SX1308` boost converter for OLED power
- USB-C input and ESD protection
- Camera and OLED FPC connectors
- Battery, speaker, motor, user button, and microphone board connectors
