# Mainboard

The Keero mainboard is the central PCB of the system. It is designed as the anchor for all compute, sensor, audio, power, and module connectivity.

## Key ICs

| Ref | Part | Role |
| --- | --- | --- |
| `U1` | `AXP2101` | PMIC, charging, rail generation |
| `U2` | `ESP32-S3-WROOM-1U-N16R8` | Main MCU with Wi-Fi/BLE |
| `U3` | `MAX98357AETE+T` | I2S audio amplifier |
| `U4` | `LIS2DW12TR` | Accelerometer |
| `U5` | `ICS-41350` | Digital MEMS microphone |
| `U6` | `DRV2605LDGSR` | Haptic driver |
| `U7` | `SX1308` | Display boost converter |

## External Connectors

| Ref | Label | Function |
| --- | --- | --- |
| `USB1` | `TYPE-C-31-M-22` | USB power and programming |
| `CN1` | `BATT` | Battery input |
| `CN2` | `CAMERA` | OV2640 FPC connector |
| `CN3` | `SPK` | Speaker output |
| `CN4` | `MOTOR` | Haptic motor connection |
| `CN5` | `OLED` | OLED FPC connector |
| `H1` | `BOTTOM_HEADER` | Bottom expansion header |
| `H2` / `H5` | `User_Button` | User button board/header |
| `H3` / `H6` | `MIC` | Microphone board/header |
| `H4` | `ESP_Header` | Additional breakout header |
| `J1` | `POGO` | Docking, charging, and serial expansion |

## Board Controls

- `SW1`: `PWRON`
- `SW2`: `RST`
- `SW3`: `BOOT`
- `SW4`: `USER BUTTON`

These controls make sense for bring-up and development: power sequencing through the PMIC, boot mode forcing, reset access, and one dedicated interaction button.

## Notes from the Schematic

- The schematic explicitly separates analog and digital domains with labels such as `AGND`, `AV_2.8V`, and `DV_1.5V`.
- Camera control is partially fixed in hardware because `CAM_RST` and `CAM_PWDN` are not under dynamic firmware control in this revision.
- USB ESD protection is handled by `D5` (`USBLC6-2SC6`).
- The charging path includes `LED1` as a visible charge indicator and the PMIC IRQ lines are exposed to the MCU.
