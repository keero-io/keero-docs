# Connector Map

This page focuses on practical integration. The current schematic clearly exposes connector names and major signal groups, but not every connector pin number can be recovered cleanly from the exported PDF text layer. For that reason, this page is written as an accurate signal map instead of an invented numbered pinout.

## USB-C

Connector:

- `USB1`
- Part: `TYPE-C-31-M-22`

Relevant signals visible in the schematic:

- `VBUS`
- `USB_D_P`
- `USB_D_N`
- `USB_P`
- `USB_N`
- `CC1`
- `CC2`
- `SBU1`
- `SBU2`
- `SHELL`

Associated support parts:

- `D5` `USBLC6-2SC6` for ESD protection
- `R1`, `R3` `5.1k` CC resistors

Use:

- firmware flashing
- development power
- PMIC input path

## OLED Connector

Connector:

- `CN5`
- Label: `OLED`
- Part: `KH-FG0.5-H2.0-30PIN`

Display-related signals visible in the schematic:

- `DISP_CS`
- `DISP_RST`
- `DISP_SCK`
- `DISP_MOSI`
- `VPP`
- `VPPA`
- `3V3B`
- `GND`

Power block associated with the OLED connector:

- `U7` `SX1308`
- `L4` `4.7uH`
- `D4` `SS14`

This indicates the display needs a boosted rail and is not driven directly from the raw battery or a simple `3V3` rail.

## Pogo Connector

Connector:

- `J1`
- Label: `POGO`

The schematic and README together indicate this interface is used for:

- charging or docking power
- UART communication
- expansion and module attachment

Because the exact pad ordering is not recoverable from the exported text alone, the safest documentation is:

- treat `J1` as the dock-facing power and serial interface
- verify final pad numbering against the original EasyEDA source before manufacturing a mating accessory
